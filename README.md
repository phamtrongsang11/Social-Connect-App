# Social Connect App
Social Connect App is a feature-rich social media application created with the Laravel PHP framework, MySQL on the backend and the Vue.js JavaScript framework, tailwind on the frontend. It provides users with a seamless and engaging social networking experience:

- User Registration and Authentication
- Profile Creation and Customization
- Management post, group
- Like, comment post
- Friends and Connections

## To run project
1. run artisan update and atisan install
3. run npm install and npm run build
4. create .env file, copy .env.example to .env and define url of database, JWT_SECRET
5. Connect to mysql server and create database
6. run php artisan migrate
